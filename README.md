# coordrounder
Coordinate rounder for plots written in Python 3

## Running

1. Run tests

```bash
$ python3 -m unittest tests/test_*
..
----------------------------------------------------------------------
Ran 2 tests in 0.001s

OK
```

2. Run script

```bash
$ python3 src/coordrounder.py
Input coord to prettify: 0.234
Pretty coord = 0.25
```
